RLC 電路之二階微分方程分析報告
一、研究背景

在工程系統中，二階微分方程比一階更具代表性，因為它能同時描述系統的慣性、儲能與振盪行為。常見應用包含電感（L）、電容（C）與彈簧-質量系統等。

本報告以RLC電路為例，分析其動態行為與輸入訊號之關係。

二、工程問題

考慮一串聯RLC電路，探討以下問題：

電路中的電流與電壓是否會產生振盪？
振盪如何隨時間衰減？
在外加電壓 V(t)=cos(t) 時，電流 i(t) 的響應為何？

七、結論

本報告透過RLC電路說明二階微分方程在工程中的重要性：

可描述振盪與能量交換
可分析阻尼效應
能預測系統對外部輸入的動態與穩態反應

RLC模型廣泛應用於：

電路設計
濾波器分析
控制系統
